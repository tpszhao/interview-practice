Given two strings `s`, find the length of the longest substring with unique characters.

```
Input: S = "abcdeef"
Output: 4
Explanation: longest such substring is "abcd"
```

```
Input: S = "aaaaaaaaaaaaaa"
Output: 1
Explanation: longest such substring is "a"
```
